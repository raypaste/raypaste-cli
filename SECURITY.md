# Security Policy

If you are concerned about a code or dependency vulnerability follow the below guidelines/rules for reporting.
Thank you for your concern in keeping open source software safe and helping protect users of Raypaste software.

## Supported Versions

| Version  | Supported          |
| -------  | ------------------ |
| 0.3.x    | :white_check_mark: |
| < 0.3.0  | :x:                |

## Reporting a Vulnerability

Report vulnerabilites by sending an email to hello@raypaste.com

Please prefix email subject with: "CODE VULNERABILITY - [repository-name]"
